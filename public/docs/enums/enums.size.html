Enumeration SIZE

Represents possible size values that can be applied to organisations.

These values are derived from the Companies Act 2006. Whilst the full definitions include considerations around annual turnover and balance sheet totals, the delineations based on the number of employees are:

  • Micro: ≤10;
  • Small: ≤50;
  • Medium: ≤250; and
  • Large: >250.
